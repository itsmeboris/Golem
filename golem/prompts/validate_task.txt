## Evaluation Criteria

1. **Task Completion**: Did the agent address all aspects of the original task?

3. **No Regressions**: Did the agent avoid breaking existing functionality?
4. **Documentation**: Did the agent document its work appropriately?
5. **Verification Evidence**: The agent should have run ``black --check .``,

Respond with ONLY this JSON structure (no other text):

Rules:
- **verdict**: One of "PASS", "PARTIAL", or "FAIL"
  - PASS: The task was fully completed with acceptable quality
  - PARTIAL: The task was partially completed; specific aspects need follow-up
  - FAIL: The task was not meaningfully completed or introduced problems
- **confidence**: REQUIRED, 0.0 to 1.0 — how confident you are in the verdict.
  Must reflect how much evidence you had to make the judgment:
  - 0.9-1.0: You can see all expected changes, tests, and they look correct
  - 0.7-0.89: Changes look correct but you cannot fully verify (e.g. diff was truncated)
  - 0.5-0.69: Partial evidence — some expected items present, others unclear
  - Below 0.5: Low evidence — verdict is largely a guess
- **concerns**: List specific, actionable items (empty list if none)
- **files_to_fix**: List specific file paths that need modification (empty list if none)
- **test_failures**: List specific failing test names or error messages from the event log (empty list if none)
- **task_type**: One of "code_change", "investigation", "documentation", "configuration", "other"

Evaluation notes:
- If there were no code changes and the task was investigative, evaluate based on documentation and tools used
- If there is a git diff, evaluate the quality and relevance of the changes
- If the agent encountered errors, weigh whether they were resolved or left unaddressed
- An empty git diff is fine for investigation/documentation tasks
- If the git diff is truncated, lower your confidence accordingly and note it in concerns
- If no verification commands (black, pylint, pytest) appear in the event log for a code_change task, lower confidence by at least 0.15 and flag in concerns
