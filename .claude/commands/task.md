Show the user a quick reference for Golem task management:

## Golem Task Commands

| Command | Description |
|---------|-------------|
| `golem run "prompt"` | Submit a new task |
| `golem run --batch file.yaml` | Submit a batch of tasks |
| `golem status` | Show daemon and task status |
| `golem dashboard` | Open the web dashboard |
| `golem logs TASK_ID` | View task logs |
| `golem stop` | Stop the daemon |

Then check if the daemon is running with `golem status` and report current state.
